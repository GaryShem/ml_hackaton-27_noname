## Описание проекта

**ChemAI: Predict the Cure** — решение хакатона(Kaggle) по предсказанию биологической активности молекул против вируса гриппа.  
На основе 192 RDKit-дескрипторов необходимо предсказать три показателя:

- **IC50** – ингибирующая концентрация
- **CC50** – цитотоксическая концентрация
- **SI** – селективность(SI = CC50 / IC50)

Метрика — средний RMSE по трём таргетам.  
Обучающая выборка: 750 соединений, тестовая: 250.

## Подход и этапы

### 1. Большой ансамбль (365 → ~278)
- Лог-трансформация таргетов
- LightGBM, XGBoost, CatBoost, два потока (log+raw)
- Stacking, multi‑trim SLSQP, pseudo‑labeling
- Diversity pack(ExtraTrees, Ridge, Huber, GaussianProcess)
- Плато ~282.6 LB после 1500+ фитов

### 2. Упрощение базы(278 → ~272)
- Оставлены только CatBoost + LightGBM (бленд 75/25)
- Удаление выбросов: `SI < 2000` → 746 строк
- Drop константных и высококоррелированных фич → 159 фич
- log1p только для SI
- Скачок до ~271.7 LB после добавления v2c стека

### 3. Multiseed и тонкая настройка (272 → 268.76)
- Усреднение 3 сидов(42, 1337, 2024) — снижение variance
- Сужение adaptive pct с 22% до **3.5%**
- Точные перцентильные клипы: IC50 на 98.80, CC50 на 98.95

## Финальный pipeline

### 1. Multiseed base
- CatBoost + LightGBM(бленд 0.75 / 0.25)
- 3 сида × 10‑fold KFold
- Удаление выбросов (SI<2000) и фич (corr>0.95) → 159 фич
- log1p только для SI

### 2. v2c SI stack
- 4 варианта LightGBM на расширенных фичах:
  - 192 дескриптора + OOF/pred IC50, CC50, SI
- 5‑fold стратификация × 3 сида

### 3. Adaptive top‑3.5% min‑of‑teachers
- Для каждой тестовой строки – log‑std предсказаний 4 учителей
- Топ 3.5% самых неуверенных → минимум из 4 учителей
- Остальные 96.5% → среднее ансамбля

### 4. Post‑processing clips
- IC50 клип на **98.80 перцентиле** test‑предсказаний
- CC50 клип на **98.95 перцентиле**

## Что сработало и не сработало

### Сработало
- Multiseed averaging(3 сида) – снижение variance
- Узкий adaptive pct(22% → 3.5%) – стабильный gain >3 LB
- Клипы IC50/CC50
- Упрощение базы(меньше моделей, меньше фич) – главный скачок
- v2c SI stack на OOF‑предсказаниях

### Не сработало
- 5‑seed averaging - регрессия
- Слишком широкий adaptive pct(>6%)
- Blends на основе derived SI
- Сложный ансамбль многих GBT – OOF‑gain не конвертился в LB
- Внешние данные ChEMBL – mismatch распределений
